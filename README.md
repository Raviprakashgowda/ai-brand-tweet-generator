# AI Brand Tweet Generator

An AI-powered tool that generates **10 on-brand tweets** based on a brand’s tone, personality, and communication style.

This project analyzes brand details and uses a large language model to generate tweets that match the brand’s social media voice.

---

## Features

- Brand voice analysis
- Automatic tweet generation
- Generates 10 tweets in brand tone
- Supports multiple tweet styles:
  - Engaging
  - Promotional
  - Witty / humorous
  - Informative

---

## Tech Stack

- Python
- Streamlit (UI)
- Groq API (Llama Model)
- Prompt Engineering
